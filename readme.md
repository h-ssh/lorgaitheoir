# lorgaitheoir

### !!! under development !!!

Artefact scanner written in Rust

## Usage

```sh
usage: lorg [scanner ...] <file>

options:
  -kw, --keyword <word>  search for keyword in file
  -e, --email            search for email addresses in file
  -h, --help             show this help
```

## Build

```sh
cargo build
```
